Heaven Delights API

Endpoints

User Endpoints
Path Method Description
/user GET Gets all existing user accounts
/user/{id} GET Gets a specific existing user account
/user/register POST Creates a new account
/user/updateuser PATCH Updates user account
/user/deleteuser/{id} DELETE Deletes a user account
/user/login POST Logs into user account
Product Endpoints
Path Method Description
/product GET Gets all products
/product/{id} GET Gets a specific product
